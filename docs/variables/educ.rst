=====================
educ: Education level
=====================

Description
===============
educ is the education code for an individual, consistently coded over
1979-present, with the exception of a break for the CPS redesign in 1994.

The recoding follows Jaeger (1997) classifications.

This variable is created in epi_basic_education.do.
